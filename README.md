# Automated Testing with Selenium
* Basic of Selenium
* Selenium with Java
* Selenium Grid
* Better testing with Selenium


Step to run with Docker compose
```
$cd selenium-grid/software/docker
$docker-compose up -d
$docker-compose scale chrome=5
```

Access to Hub via browser  http://localhost:444
